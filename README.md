### Department of AIML & Data Science Website

This project is a fully responsive, modern frontend website for the Department of Artificial Intelligence, 
Machine Learning & Data Science. It showcases the department’s programs, faculty, research initiatives, and 
provides contact and quick-access resources for students and visitors.

The website is designed with a clean, professional aesthetic using HTML5, CSS3, and JavaScript, with animations and interactive 
components for an engaging user experience. It is optimized for both desktop and mobile devices.

### Features

Responsive Design: Fully adapts to desktop, tablet, and mobile screens.

Animated Hero Section: Interactive hero banner with smooth animations and call-to-action buttons.

Navigation:

Fixed header with scroll effects.

Mobile-friendly collapsible menu.

Smooth scrolling to sections.

About Section: Highlights department history, mission, and key statistics.

Academic Programs: Detailed cards for B.Tech programs in AI & ML, Data Science, and ECE (AIML).

Faculty Profiles: Showcase of faculty members with photos, roles, bios, and social links.

Research & Innovation: Overview of research areas and student involvement in cutting-edge projects.

Contact Form: Collects messages from visitors along with department contact information.

Quick Links Sidebar: Easy access to academic calendar, syllabus, student portal, placement cell, and more.

Footer Section: Detailed information, program links, resources, and social media integration.

Interactive UI Enhancements:

Gradient text headings and buttons.

Card hover effects with shadows.

Tagline and research area badges with hover animations.

Smooth scroll animations and pulsing elements for engagement.

### Technologies Used

HTML5 – Semantic markup for content structure.

CSS3 – Modern styling including gradients, flexbox, grid layout, animations, and responsive design.

JavaScript – Interactivity such as mobile menu toggle, smooth scroll, and dynamic header styling.

Font Awesome – Icons for programs, faculty, and social media.

Google Fonts – Inter fonts (Inter and Poppins) for typography.

### Installation & Usage

Clone the repository

git clone https://github.com/yourusername/aiml-department-website.git


Open index.html in a web browser to view the website.

Optional: Serve with a local development server for live reload.

npx live-server

### Customization

Update faculty profiles by editing .faculty-card sections in index.html.

Add new programs or research areas by modifying .programs-grid and .research-areas.

Change color scheme via CSS variables in :root.

Replace placeholder images/icons with actual department assets.
